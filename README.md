# RestfyScanner

**Installing application**

```
  // Downloading project
  git clone https://github.com/lucasGabrielDeAA/RestfyScanner.git && cd RestfyScanner

  // Installing dependencies
  yarn

  // For Android's environment
  yarn android

  // For IOS's environment
  cd ios/ && pod install && cd .. && yarn ios
```
